Sobre o MuscleMestre

O MuscleMestre é uma plataforma web desenvolvida para ajudar você a explorar e encontrar os melhores exercícios para o seu treino, com base na API pública da WGER.

Aqui você pode:

  • Filtrar exercícios por grupo muscular, categoria ou equipamento
  • Visualizar imagens e descrições detalhadas dos exercícios
  • Favoritar exercícios para acesso rápido depois

Esta aplicação foi feita como um projeto educacional e está em constante evolução.
